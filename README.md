# Ethereum Block Explorer

A lightweight Ethereum block explorer inspired by Etherscan, built with **React** and **Alchemy SDK**.

## Features

- View latest Ethereum blocks
- Browse recent blocks and transactions
- Transaction detail pages (from, to, value, gas, status)
- Account lookup (ETH balance, tx count, recent transfers)
- Click-through navigation: Block → Transaction → Account

## Tech Stack

- React
- Alchemy SDK (ethers.js compatible)
- Ethereum Mainnet
- React Router
- Custom CSS
